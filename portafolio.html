Servi - Apart

Junto con mi equipo de trabajo en el SENA, desarrollamos como proyecto final un sitio web integral utilizando tecnologías como PHP, CSS, JavaScript y Bootstrap. El objetivo principal fue simplificar las operaciones diarias en conjuntos residenciales, abarcando desde el registro de usuarios, vehículos y posibles daños, hasta la gestión de publicaciones, paquetería y reservas para el salón comunal. Este sitio proporciona una solución completa para administradores, residentes y guardias de seguridad, permitiendo una integración fluida de funcionalidades que mejora la eficiencia global en la gestión de conjuntos residenciales.

Chat en tiempo real

Desarrollé un emocionante chat en tiempo real utilizando JavaScript y Service Workers para ofrecer una experiencia de comunicación instantánea entre dos usuarios simulados. En esta simulación, una ventana representa al Usuario 1 y la otra al Usuario 2. La aplicación permite que los usuarios intercambien mensajes de manera rápida y eficiente, con una interfaz intuitiva que facilita la visualización y el envío de mensajes en tiempo real.

Ejercicio Interactivo con CSS

He desarrollado un ejercicio interactivo, realizado exclusivamente con CSS, destacando el uso de nth-child para la disposición y estilo de cada elemento. Este proyecto demuestra la versatilidad de CSS en la creación de experiencias visuales dinámicas y atractivas, sin necesidad de utilizar otros lenguajes de programación.

Simulador de Inversión

He diseñado un simulador de inversión bancaria que ofrece a los usuarios la oportunidad de proyectar sus ganancias en función del tiempo y la tasa de interés. Este ejercicio interactivo proporciona una experiencia intuitiva donde los usuarios pueden ajustar variables como el plazo de inversión y la tasa de interés para visualizar de manera dinámica cómo evolucionaría su dinero con el tiempo.
